Sobre o DJEN Consultor

Uma ferramenta moderna e eficiente para consulta de comunicações judiciais

O que é?

O DJEN Consultor é uma aplicação web gratuita que permite consultar múltiplas comunicações processuais através da API pública do Diário de Justiça Eletrônico Nacional (DJEN).

Desenvolvida para facilitar o trabalho de advogados, procuradores, defensores públicos e demais operadores do direito, a ferramenta automatiza o processo de consulta, permitindo verificar várias intimações, editais e outras comunicações de forma rápida e organizada.

Funcionalidades

  • Consulta em lote de múltiplos processos simultaneamente
  • Extração automática de números de processo do texto colado
  • Filtros avançados por data, tribunal, órgão e tipo de comunicação
  • Interface responsiva com suporte a modo claro e escuro
  • Visualização organizada dos resultados em cards informativos
  • Exportação de dados em formato JSON
  • Estatísticas detalhadas sobre as consultas realizadas
  • Links diretos para os documentos originais

Tecnologias Utilizadas

CSS3

API DJEN

Privacidade e Segurança

O DJEN Consultor respeita sua privacidade. Todas as consultas são realizadas diretamente do seu navegador para a API pública do DJEN, sem passar por servidores intermediários.

Nenhum dado de processo ou resultado é armazenado em nossos servidores. As informações permanecem apenas no seu navegador e podem ser excluídas a qualquer momento.
