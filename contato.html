Perguntas Frequentes

Como funciona a consulta de processos?
A ferramenta extrai automaticamente os números de processo do texto que você colar, consulta a API pública do DJEN e exibe os resultados de forma organizada. Você pode filtrar por data, tribunal e tipo de comunicação.
Os dados consultados ficam armazenados?
Não. Todas as consultas são feitas diretamente do seu navegador para a API do DJEN. Nenhum dado é enviado ou armazenado em nossos servidores. As informações ficam apenas no seu navegador.
Existe limite de consultas?
O limite de consultas é determinado pela API pública do DJEN. Geralmente, há um rate limit para evitar sobrecarga. A ferramenta adiciona um intervalo de 1 segundo entre cada consulta para respeitar esses limites.
Posso contribuir com o projeto?
Sim! O projeto é de código aberto. Você pode contribuir com código, documentação, reportar bugs ou sugerir melhorias através do GitHub. Toda contribuição é bem-vinda!
A ferramenta funciona em dispositivos móveis?
Sim! A interface é totalmente responsiva e funciona em smartphones, tablets e desktops. O layout se adapta automaticamente ao tamanho da tela do dispositivo.
